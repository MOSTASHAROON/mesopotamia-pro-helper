== Description ==

> This Plugin requires [Mesopotamia theme](https://mostasharoon.org/mesopotamia/).
> We don't give any support to this plugin here, if you need any help please feel free to contact us through our [support page](http://mostasharoon.org/support/).

Mesopotamia Pro Helper allows you to easily integrate many premium plugins like UberMenu and Slider Revolution with Mesopotamia theme.

== Installation ==

1. Upload the entire `mesopotamia-pro-helper` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
